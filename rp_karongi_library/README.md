# RP Karongi Library - Local PHP/MySQL App

Place this folder inside your XAMPP `htdocs` (e.g. `C:\xampp\htdocs\rp_karongi_library`) and:
1. Start Apache and MySQL in XAMPP.
2. Import `sql/create_db.sql` using phpMyAdmin (http://localhost/phpmyadmin).
3. Replace the placeholder Bootstrap files in `assets/css/bootstrap.min.css` and `assets/js/bootstrap.bundle.min.js` with official offline Bootstrap files (download separately) for full Bootstrap features.
4. Open: http://localhost/rp_karongi_library/index.php

Default DB credentials are in `inc/config.php` (root, no password).
